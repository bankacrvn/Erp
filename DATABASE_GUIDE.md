## Row Level Security

### Enable RLS

```sql
-- Enable RLS on table
ALTER TABLE orders ENABLE ROW LEVEL SECURITY;

-- Create policy: Users can only see their own orders
CREATE POLICY "Users can view their own orders" ON orders
  FOR SELECT
  USING (auth.uid() = user_id);

-- Create policy: Users can insert their own orders
CREATE POLICY "Users can insert their own orders" ON orders
  FOR INSERT
  WITH CHECK (auth.uid() = user_id);
```

## Common Operations

### 1. ‡∏™‡∏£‡πâ‡∏≤‡∏á Order ‡∏û‡∏£‡πâ‡∏≠‡∏° Items

```typescript
async function createOrderWithItems(orderData, items) {
  // 1. ‡∏™‡∏£‡πâ‡∏≤‡∏á order
  const { data: order, error: orderError } = await supabase
    .from('orders')
    .insert([orderData])
    .select()
    .single();

  if (orderError) throw orderError;

  // 2. ‡∏™‡∏£‡πâ‡∏≤‡∏á order items
  const itemsWithOrderId = items.map(item => ({
    ...item,
    order_id: order.id
  }));

  const { data: orderItems, error: itemsError } = await supabase
    .from('order_items')
    .insert(itemsWithOrderId)
    .select();

  if (itemsError) throw itemsError;

  return { order, orderItems };
}
```

### 2. ‡∏õ‡∏¥‡∏î‡∏Å‡∏∞ (Close Shift)

```typescript
async function closeShift(shiftId, closingBalance) {
  // 1. ‡∏î‡∏∂‡∏á shift data
  const { data: shift } = await supabase
    .from('shifts')
    .select('*')
    .eq('id', shiftId)
    .single();

  // 2. ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì total sales
  const { data: payments } = await supabase
    .from('payments')
    .select('amount')
    .gte('created_at', shift.open_time)
    .lte('created_at', new Date());

  const totalSales = payments.reduce((sum, p) => sum + p.amount, 0);
```

```typescript
async function getDailyReport(date) {
  // 1. ‡∏î‡∏∂‡∏á orders ‡∏Ç‡∏≠‡∏á‡∏ß‡∏±‡∏ô‡∏ô‡∏±‡πâ‡∏ô

  const totalOrders = orders.length;

    .filter(p => p.payment_method === 'cash')
    .reduce((sum, p) => sum + p.amount, 0);
  const cardPayments = payments
    .filter(p => p.payment_method === 'card')
    .reduce((sum, p) => sum + p.amount, 0);

    totalOrders,
    totalSales,
```
